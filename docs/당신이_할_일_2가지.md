# 당신이 할 일 — 딱 2가지

나머지는 이미 코드에 다 넣어 두었어요. **아래 2가지만 하면 됩니다.**

---

## 1. Supabase 웹사이트에서 SQL 한 번 실행하기

1. 브라우저에서 **https://supabase.com** 들어가서 **로그인**.
2. 만든 **프로젝트** 들어가기 (이름 클릭).
3. 왼쪽 메뉴에서 **「SQL Editor」** 클릭.
4. **「New query」** 버튼 누르기.
5. 이 프로젝트 폴더 안에 있는 **`docs/SUPABASE_SCHEMA.sql`** 파일을 연 다음, **안에 있는 글자 전부** 복사 (Cmd+A → Cmd+C).
6. Supabase SQL Editor 안 **빈 칸에 붙여넣기** (Cmd+V).
7. 오른쪽 위 **「Run」** 버튼 누르기.
8. 에러 없이 끝나면 **1번 끝**.

---

## 2. 프로젝트에 `.env` 파일 만들고 값 넣기

1. Cursor 왼쪽 **파일 트리**에서 프로젝트 **맨 바깥 폴더** (뷰티박스 웹사이트 폴더) 확인.
2. 그 폴더에 **새 파일** 만들기. 이름은 **`.env`** (앞에 점 붙이고, 뒤는 env).
3. 그 파일 안에 **아래 2줄** 넣고 저장.  
   (두 번째 줄 `여기에...` 부분은 **Supabase에서 받은 Publishable key**로 바꾸기.  
Supabase 대시보드 → 왼쪽 아래 **Settings** (톱니바퀴) → **API** → **Project API keys** 에서 **anon / public** 키 복사해서 쓰면 됨.)

```
VITE_SUPABASE_URL=https://jdddyczgycxeclwchjin.supabase.co
VITE_SUPABASE_ANON_KEY=여기에_본인_키_붙여넣기
```

4. **터미널**에서 한 번만 다시 실행:

```
npm run dev
```

5. 이렇게 하면 **2번도 끝**.

---

## 정리

- **1번:** Supabase 사이트 → SQL Editor → `SUPABASE_SCHEMA.sql` 내용 복사 → 붙여넣기 → Run  
- **2번:** 프로젝트 루트에 `.env` 만들고 URL이랑 키 넣기 → `npm run dev` 다시 실행  

이 2가지만 하면, 나머지는 이미 코드에 다 되어 있어요.
