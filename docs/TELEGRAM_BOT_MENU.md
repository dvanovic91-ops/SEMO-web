# Telegram user-бот (`My_SEMO_Beautybot`) — меню

**Код самого бота в этом репозитории нет.** Ниже — что уже сделано **на сервере (Supabase)** и какое **меню** нужно в боте. Правки кнопок и текстов делаются **в папке с кодом бота** (Python/Node и т.д.).

---

## 지금까지 이 웹·저장소에서 한 일 (서버 쪽)

- **DB에 규칙(SQL)** 넣으면: `telegram_id`로 **주문** / **쿠폰·포인트·배송** 같은 **웹과 같은 데이터**를 가져올 수 있게 해 둠.  
  파일 예: `docs/SUPABASE_RPC_TELEGRAM_ORDERS_PREVIEW.sql`, `docs/SUPABASE_RPC_TELEGRAM_CABINET.sql` (대시보드 SQL에서 실행한 적 있어야 함.)
- **인터넷 주소(Edge)** 로 부르면 위 데이터를 **텔레그램 글자**로 받을 수 있게 해 둠:  
  `telegram-orders-preview`, `telegram-cabinet-preview` (이미 배포했다면 OK.)
- **비밀 문자 + 쇼핑몰 주소**는 Supabase Secrets에 넣어 둔 것.

**봇 메뉴 글자·버튼 순서**는 서버가 아니라 **봇 코드**에서 바꿉니다. 이 파일은 그 **설계도**입니다.

## Главное меню (4 кнопки в один ряд или 2×2)

| № | По-русски в боте | Что делает |
|---|------------------|------------|
| 1 | **Тест кожи** | Как сейчас: сценарий теста кожи (к сайту / внутри бота). |
| 2 | **Магазин** | Витрина / товары (не смешивать с личным кабинетом). |
| 3 | **Личный кабинет** | Открывает **подменю** ниже. |
| 4 | **Помощь** | FAQ / поддержка / ссылка на сайт — как решите. |

*(«Помощь» — это ваш «뽀모싀» / помощь.)*

---

## Личный кабинет — подменю (без «гид по уходу»)

**Убрать:** пункт **«Гид по уходу»** (или любой такой же) из раздела личного кабинета — **не показывать**.

**Показывать по кнопкам:**

| Кнопка (рус.) | Смысл | Откуда данные (сервер уже есть) |
|---------------|--------|----------------------------------|
| **Результаты теста** | Последний тип кожи / ссылка на сайт | `skin_test_results` или страница `/profile/test-results` |
| **Мои заказы** | Сюда перенести то, что раньше было отдельным «заказы» | Edge **`telegram-orders-preview`** или RPC `telegram_orders_preview` → текст `message_ru` |
| **Баллы** | Баланс, как на сайте | Edge **`telegram-cabinet-preview`** (в ответе есть `points`) или весь блок кабинета |
| **Купоны** | Список купонов | Тот же **`telegram-cabinet-preview`** (в ответе есть `coupons`) |
| **Назад** | Вернуть **главное меню** из 4 кнопок | Просто снова показать клавиатуру главного экрана |

Если нужно **одним сообщением** «и баллы, и купоны, и адрес» — можно вызывать только **`telegram-cabinet-preview`** и для кнопок «Баллы» / «Купоны» **обрезать текст** или показывать разные части одного ответа.

## Проверка

- После смены меню в боте: нажать **Личный кабинет** → нет «гида по уходу», есть 5 пунктов + **Назад** → главное меню снова 4 кнопки.
