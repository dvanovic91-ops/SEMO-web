# 관리자 상품 이미지·설명 업로드 방법

관리자 모드에서 **제품 썸네일 1장**, **상세 설명**, **상세 페이지 구성품 사진**을 어디에 넣으면 어떻게 반영되는지 정리한 문서입니다.

---

## 1. 반영 위치 요약

| 넣는 곳 (관리자 폼) | 반영되는 화면 |
|---------------------|----------------|
| **① 썸네일·상세 페이지 대표 이미지 (1장)** | 상품 상세 페이지 상단 큰 사진. 쇼핑/목록에서 해당 상품을 쓸 때 대표 이미지로도 사용됩니다. |
| **② 상세 설명 (본문)** | 상품 상세 페이지 본문 영역에 그대로 표시됩니다. (HTML/텍스트) |
| **③ 상세 페이지 구성품 이미지 (최대 6개)** | 상품 상세 페이지 하단 구성품 그리드. 각 항목의 이름·이미지·설명이 표시됩니다. |

---

## 2. 업로드 방법 (두 가지)

### 방법 A: URL 입력

- 이미지를 다른 곳(이미지 호스팅, Supabase Storage 등)에 올린 뒤 **공개 URL**을 복사해서 해당 입력란에 붙여넣기합니다.
- **① 대표 이미지**: URL 입력란에 `https://...` 형식으로 입력.
- **③ 구성품**: 각 항목의 "이미지 URL" 란에 입력.

### 방법 B: 파일 올리기 (권장)

- 관리자 페이지에서 **「파일 올리기」** 버튼을 누르고, PC에서 이미지 파일을 선택하면 **Supabase Storage(`product-images` 버킷)**에 자동 업로드되고, 입력란에 URL이 채워집니다.
- **① 대표 이미지**: "파일 올리기" 클릭 → 파일 선택 → 업로드 완료 후 URL 자동 입력.
- **③ 구성품**: 각 항목 오른쪽 "파일 올리기" 클릭 → 파일 선택 → 해당 항목 이미지 URL 자동 입력.

**※ 파일 올리기를 쓰려면** Supabase에서 **`product-images`** 버킷을 미리 만들어 두어야 합니다.  
→ 자세한 설정: [docs/SUPABASE_STORAGE.md](./SUPABASE_STORAGE.md) 의 「관리자 상품 이미지 버킷」 섹션을 참고하세요.

---

## 3. 진행 순서 (한 번에 하기)

1. **관리자 모드** 접속 후 **상품** 탭 선택.
2. 기존 상품을 선택하거나 **새 상품 추가** 후, 상품명·가격 등 기본 정보 입력.
3. **① 썸네일·대표 이미지**  
   - URL을 알고 있으면 입력란에 붙여넣기.  
   - 모르면 **「파일 올리기」** → 이미지 선택 → 저장 후 자동으로 URL 반영.
4. **② 상세 설명**  
   - 상품 상세 페이지에 보일 본문 내용을 입력 (HTML/텍스트).
5. **③ 구성품**  
   - 「+ 항목」으로 1~6개 추가 후, 각 항목에 **이름**, **이미지**(URL 입력 또는 「파일 올리기」), **설명** 입력.
6. 맨 아래 **저장** 버튼 클릭.

저장이 끝나면 상품 상세 페이지에 **대표 이미지**, **상세 설명**, **구성품 이미지**가 위 표와 같이 반영됩니다.
