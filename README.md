# Beauty Box Korea → Russia

React + TypeScript + Vite + Tailwind CSS 기반의 뷰티박스 수출(러시아) 서비스 웹사이트 뼈대입니다.

## Tech stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router

## 개발 환경 세팅

```bash
npm install
npm run dev
```

기본 포트는 `5173`입니다.

## 주요 페이지

- `/` : 랜딩 페이지
- `/skin-test` : Bauman 16 피부 타입 테스트 페이지(폼 예정 영역 표시)
- `/shop` : 시즌 패키지 및 상품 리스트 뼈대
- `/support` : 구매 방법 및 고객 지원 안내 뼈대

상단 네비게이션 바 우측에 항상 개인정보/마이페이지로 이어질 수 있는 프로필 아이콘이 배치되어 있습니다.
